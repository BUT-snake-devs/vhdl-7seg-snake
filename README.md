# VHDL 7-Segment Snake 🐍
University team project: Snake game logic written in VHDL using a 7-segment display output.

## 👥 About the Project
This is a university team project. We implemented the classic Snake game logic using VHDL. The game output is rendered entirely on a 7-segment display. 

## 🛠️ Tools & Technologies
* **Language:** VHDL
* **Development Environment:** Xilinx Vivado / VS Code
* **Target Board:** Nexys A7-50T
